# Encrypted Notes
Encrypted Notes uses RSA encryption to securely save **your** text securely on your computer so you can be sure your data/text is safe.

## - Features
- RSA Encryption
- RSA Key Regeneration
- *Some what sleek* UI
- Low memory usage
